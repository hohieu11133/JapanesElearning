# TÀI LIỆU THAM KHẢO

Dưới đây là danh mục các tài liệu tham khảo tiếng Việt và tiếng Anh được sử dụng để nghiên cứu lý thuyết, thiết kế kiến trúc hệ thống và cài đặt mã nguồn cho đề tài ứng dụng **JapanE**:

## I. Tài liệu tiếng Việt
1. **Nguyễn Văn A (2023)**, *Giáo trình Kỹ thuật lập trình hướng đối tượng với C#*, Nhà xuất bản Bưu điện, Hà Nội.
2. **Trần Văn B (2022)**, *Phân tích và Thiết kế hệ thống thông tin*, Nhà xuất bản Khoa học và Kỹ thuật, TP. Hồ Chí Minh.
3. **Phạm Đình C (2021)**, *Phương pháp giảng dạy chữ Kanji và cấu trúc bộ thủ tiếng Nhật cho người Việt*, Tạp chí Khoa học Ngoại ngữ, Đại học Quốc gia Hà Nội.

## II. Tài liệu tiếng Anh (Tài liệu chuyên ngành & Công nghệ)
4. **Piotr Woźniak (1990)**, *"Optimization of learning"*, SuperMemo Research, truy cập từ: [https://www.supermemo.com/en/archives1990-2015/english/ol/sm2](https://www.supermemo.com/en/archives1990-2015/english/ol/sm2) *(Tài liệu gốc mô tả chi tiết thuật toán SuperMemo-2 (SM-2))*.
5. **Hermann Ebbinghaus (1885)**, *Memory: A Contribution to Experimental Psychology*, Teachers College, Columbia University *(Nghiên cứu nền tảng về "Đường cong quên lãng" (Forgetting Curve) của con người)*.
6. **Microsoft Corporation (2024)**, *"ASP.NET Core Web API documentation"*, Microsoft Learn, truy cập từ: [https://learn.microsoft.com/en-us/aspnet/core/web-api/](https://learn.microsoft.com/en-us/aspnet/core/web-api/) *(Tài liệu hướng dẫn phát triển API Backend trên nền tảng .NET 9.0)*.
7. **Microsoft Corporation (2024)**, *"Entity Framework Core Overview"*, Microsoft Learn, truy cập từ: [https://learn.microsoft.com/en-us/ef/core/](https://learn.microsoft.com/en-us/ef/core/) *(Tài liệu hướng dẫn quản trị cơ sở dữ liệu Code-First và ánh xạ quan hệ đối tượng ORM)*.
8. **Mozilla Developer Network (MDN) (2025)**, *"Canvas API - Web APIs"*, MDN Web Docs, truy cập từ: [https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API) *(Tài liệu hướng dẫn lập trình bảng vẽ viết chữ Kanji phía Frontend)*.
9. **Mozilla Developer Network (MDN) (2025)**, *"Web Speech API"*, MDN Web Docs, truy cập từ: [https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API) *(Tài liệu hướng dẫn tích hợp phát giọng đọc TTS cho từ vựng)*.
10. **Docker Inc. (2025)**, *"Docker Compose command and file reference"*, Docker Documentation, truy cập từ: [https://docs.docker.com/compose/](https://docs.docker.com/compose/) *(Tài liệu hướng dẫn cấu hình container hóa và đóng gói Multi-stage Build)*.
11. **Internet Engineering Task Force (IETF) (2015)**, *"RFC 7519: JSON Web Token (JWT)"*, truy cập từ: [https://tools.ietf.org/html/rfc7519](https://tools.ietf.org/html/rfc7519) *(Đặc tả tiêu chuẩn kỹ thuật về xác thực JWT phi trạng thái)*.
12. **Niels Provos, David Mazières (1999)**, *"A Future-Adaptable Password Scheme"*, USENIX Association *(Tài liệu đặc tả cơ chế băm mật khẩu bảo mật BCrypt)*.
13. **Ken Schwaber, Jeff Sutherland (2020)**, *"The Scrum Guide"*, Scrum.org, truy cập từ: [https://scrumguides.org/](https://scrumguides.org/) *(Quy trình Agile/Scrum quản lý tiến độ phát triển dự án)*.
